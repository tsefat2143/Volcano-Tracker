# Volcano Tracker Application  
### This app tracks volcanoes and plots them on Google Maps  
  
### npm install   
### npm start  
  
# Build  
### npm build or yarn build  
  
### API used for plotting volcanoes: https://eonet.sci.gsfc.nasa.gov/api/v2.1/events
